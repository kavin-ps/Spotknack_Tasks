HTML TAGS

1)BOLD TAG

DESCRIPTION:

Bold tag is used by specifying the start and the end of the text block we want to make bold

EXAMPLE:

 Once upon a time there lived a ghost

2)BLOCK QUOTE TAG

DESCRIPTION:

when a quotation is presented inline with the surrounding text, but when the quotation is presented as a separate
paragraph, blockquote is the appropriate tag to use to identify the quotation.

EXAMPLE:

once upon a time there lived a ghost

3)BIG TAG

DESCRIPTION:

The BIG TAG was used to cause the selected text to appear one size larger than the surrounding text

EXAMPLE:

 once upon a time there lived a ghost

4)BREAK TAG

DESCRIPTION:

The break tags are very helpful to write any content with line breaks. For example, writing a Poem or an address.
In these type of content, the line break is very significant, so Break Tag helps to produce those line breaks
without changing the paragraph.

EXAMPLE:

once upon a time
there lived a ghost
